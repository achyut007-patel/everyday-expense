# Everyday Expense Tracker (College Project)

A lightweight, high-performance Single Page Application (SPA) designed to help individuals monitor their personal finances, record cash flows, and visualize categorical asset distribution in real-time.

## 🌟 Key Technical Features
* **Unified Authentication Flow:** Secure, responsive simulation matrices handling direct Login, Reset Password routines, and simulated OTP (One-Time Password) generation modules.
* **Persistent Local Ledger:** Leverages the native HTML5 Web Storage API (`localStorage`) to execute persistent read/write operations, maintaining user state data across page reloads without requiring an external database cluster.
* **Dynamic Graphical Analytics:** Interactive data mapping using **Chart.js** to generate live bar graphs tracking categorical expenditure profiles.
* **Responsive Architecture:** Styled entirely via localized, modular CSS definitions offering seamless layout switches between desktop viewports and mobile display frames.

## 🛠️ Technology Stack Utilized
* **Frontend Core:** HTML5, CSS3, Modern JavaScript (ES6+)
* **Data Visualization Library:** Chart.js (via CDN implementation)
* **Iconography assets:** FontAwesome open-source catalog
* **Hosting Environment:** GitHub Pages Infrastructure
